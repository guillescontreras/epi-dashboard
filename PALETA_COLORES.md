# Paleta de Colores CoironTech
## Inspirada en el Coirón Patagónico

### Colores Principales
- **Dorado Coirón**: `#D4A574` - Color principal del coirón
- **Ocre Tierra**: `#B8956A` - Tonos tierra de la Patagonia
- **Verde Estepa**: `#7A9B76` - Verde de la vegetación patagónica
- **Gris Piedra**: `#8B9A9F` - Gris de las rocas patagónicas
- **Azul Cielo**: `#5B8FA3` - Azul del cielo patagónico

### Colores Secundarios
- **Dorado Claro**: `#E8C9A0`
- **Verde Oscuro**: `#5A7A57`
- **Gris Oscuro**: `#4A5568`

### Gradientes Sugeridos
- Header/Footer: `from-[#8B9A9F] via-[#7A9B76] to-[#5B8FA3]`
- Botones principales: `from-[#D4A574] to-[#B8956A]`
- Botones secundarios: `from-[#7A9B76] to-[#5A7A57]`
- Alertas: `from-[#D4A574] to-[#E8C9A0]`

### Uso Recomendado
- **Primario**: Dorado Coirón para acciones principales
- **Secundario**: Verde Estepa para navegación
- **Neutro**: Gris Piedra para fondos
- **Acento**: Azul Cielo para destacados
